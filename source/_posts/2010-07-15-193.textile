（1）静态化规则是在配置文件中写入：
{% codeblock %}location / {
		if (-f $request_filename/index.html){
		        rewrite (.*) $1/index.html break;
		}
		if (-f $request_filename/index.php){
		        rewrite (.*) $1/index.php;
		}
		if (!-f $request_filename){
		        rewrite (.*) /index.php;
		}
	}{% endcodeblock %}
{% codeblock %}（2）重启nginx,进入wordpress后台。在设置中，设置固定链接，自定义结构中填写/%year%/%monthnum%/%post_id%.html{% endcodeblock %}
{% codeblock %}固定链接支持的命令参数：{% endcodeblock %}
{% codeblock %}%year% —— 文章发表年份
%monthnum% —— 文章发表月份
%day% —— 文章发表日期
%hour% —— 文章发表时间（小时）
%minute% —— 文章发表时间（分钟）
%second% —— 文章发表时间（秒）
%postname% —— 文章标题,中文显示是标题别名
%post_id% —— 文章编号（唯一性）
%category% —— 文章分类，中文显示分类别名
%tag% —— 标签名
%author% —— 作者{% endcodeblock %}
{% codeblock %}（3）OK，搞定了。{% endcodeblock %}
